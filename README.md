# james-dotfiles

One repo. Three machines. Three jobs. Everything syncs.

## What this does

- **Shell config** — aliases, functions, env vars across all Macs
- **Claude Code settings** — global CLAUDE.md, per-job attribution rules, permissions
- **Cursor settings** — synced via symlink since Cursor has no native sync
- **Job profiles** — switch context between startup/mobile/personal with one command

## Setup (run once per machine)

```bash
git clone git@github.com:YOUR_USERNAME/james-dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x scripts/install.sh scripts/sync.sh
./scripts/install.sh
```

The installer will:
- Install Oh My Zsh (if missing)
- Install Powerlevel10k theme + Meslo Nerd Font
- Symlink shell config, Claude Code CLAUDE.md, and Cursor settings
- Copy Claude Code settings.json (so `job()` can merge overrides into it)
- Back up any existing files before replacing them

**After install:** Set your terminal font to `MesloLGS NF`, then restart your terminal. P10k's config wizard will launch automatically.

## Daily usage

```bash
# Switch job context (updates git config, Claude attribution, etc.)
job crunchyroll  # Crunchyroll
job mcd          # MCD
job orka         # Orka
job personal     # Personal projects

# Pull latest config from another machine
dotpull

# Push config changes to repo
dotpush "added new alias"

# Check current job
job
```

## Machine-specific overrides

Create `~/.zshrc.local` (gitignored) for per-machine stuff:

```bash
# Example: auto-set job on startup laptop
export CURRENT_JOB="crunchyroll"

# Example: extra PATH for this machine
export PATH="$HOME/some-tool/bin:$PATH"
```

## What stays local (not in repo)

- `~/.p10k.zsh` — Powerlevel10k config (terminal-specific, generated by `p10k configure`)
- `~/.zshrc.local` — machine-specific overrides
- `~/.claude/settings.json` — copied (not symlinked) so `job()` can merge overrides
